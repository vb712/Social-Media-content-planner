# Project: Social Media Content Planner

## Goal
Build a simple web-based content planner using:
- Python (Flask)
- Excel as database (pandas + openpyxl)
- HTML/CSS frontend

## Rules
- Keep code simple and beginner-friendly
- Avoid unnecessary complexity
- Use modular structure
- No authentication required
- Focus on core features first

## Features to implement
- Add content
- View content
- Filter content
- Update status
- Weekly calendar
- Upcoming posts

## UI Requirements
- Clean and minimal
- Bootstrap allowed
- Table-based layout

## Architecture
- Flask routes handle requests
- planner.py handles business logic
- Excel file used as storage

## Output expectations
- Working code
- No errors
- Clean folder structure
